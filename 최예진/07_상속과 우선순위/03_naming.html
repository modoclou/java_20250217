클래스 선택자

아이디 선택자는 중복 사용이 불가능하고
태그 선택자는 스타일을 적용한 후 마크업이 수정되면 다시 수정해야 한다.
따라서 css 선택자로 클래스 선택자 사용을 권장한다.

  • 클래스 선택자의 장점
    1. 유지보수가 쉽다.
    2. 코드의 재사용성이 쉽다.
    3. 확장성이 높다.
