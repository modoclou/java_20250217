for 반복문

for(초기식; 조건식; 증감식) { 실행할 문장; }

  1. 초기식
  2. 조건식 확인
  3. 조건식 결과가 true면 { }안에 있는 문장 수행
  4. 증감식
  5. 2, 3, 4 반복하다가
  6. 조건식 결과가 false면 반복문 빠져 나옴

중첩 for문

for문 안에 또 다시 for문을 중첩할 수 있다.

for(초기식1; 조건식1; 증감식1){ for(초기식2; 조건식2; 증감식2){ 실행할 문장; } }

  1. 초기식1
  2. 조건식1 확인
  3. 조건식1 결과가 true면 { } 문장 수행
    1. 초기식2
    2. 조건식2 확인
    3. 조건식2 결과가 true면 { } 문장 수행
    4. 증감식2
    5. b, c, d 반복하다가
    6. 조건식2 결과가 false면 안쪽에 있는 반복문 빠져 나옴
  4. 증감식1
  5. B, C, D 반복하다가
  6. 조건식1 결과가 false면 바깥쪽에 있는 반복문 빠져 나옴
