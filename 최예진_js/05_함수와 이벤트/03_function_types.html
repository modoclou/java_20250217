함수 종류

가장 기본적인 함수 선언 방식과 호출

  • 함수 선언과 함수 호출 순서는 상관없다.

// 함수 호출(호이스팅) 함수명(); // 함수 선언 function 함수명() { }; // 함수 호출 함수명();

즉시 실행 함수

  • 함수는 선언하고 호출해서 실행하지만
  • 한 번만 실행하는 함수는 함수를 선언하면서 동시에 실행할 수 있다.
  • 한 번 실행되고 다시 호출할 수 없음

(function() { }()); 또는 (function(매개변수){ }(인수));

익명 함수

  • 함수명 없이 선언
  • 변수에 저장하여 사용
  • 호이스팅 불가능 : 선언 전에 호출하면 오류
  • 변수이므로

// 익명 함수 선언 const 변수 = function(){ } // 함수 호출 변수명();

화살표 함수

  • 함수 이름이 없을 경우 => 표기법을 사용해 함수 선언
  • 함수를 간결하게 표현하는 방식
  • function 키워드를 사용하지 않고 짧게 작성 가능

// 매개 변수 없는 경우 const sayYe = function(){ return console.log("안녕"); } const sayYe = () => { return "안녕" }; const sayYe = () => { "안녕" }; const sayYe = () => "안녕"; // 매개 변수 1개 있는 경우 const sayHello = function(user) { document.write( user + "님 안녕" ); } const sayHello = (user) => { document.write( user + "님 안녕" ) }; const sayHello = user => document.write( user + "님 안녕" ); // 매개 변수 2개 있는 경우 const sayYo = function(user, age) { document.write( user + "님은" + age + "세 입니다." ); } const sayYo = (user, age) => document.write( user + "님은" + age + "세 입니다." );

콜백 함수

  • 함수의 인자로 다른 함수를 전달하는 방식
  • 특정 작업이 완료된 후 실행할 동작을 정의할 때 사용
  • 비동기 처리 (setTimeout, 이벤트 핸들러 등)에서 많이 사용됨
