변수의 범위

  • 변수의 Scope(스코프, 유효 범위) 란 변수가 어디까지 접근할 수 있는지를 결정하는 개념
  • 자바스크립트에서는 변수가 선언된 위치에 따라 접근 가능한 범위가 달라진다.

전역변수

  • 함수나 블록({ }) 외부에서 선언된 변수
  • 프로그램 전체에서 접근 가능 (어디서든 사용 가능)
  • 하지만 전역 변수 사용은 되도록 최소한으로 사용하는 것이 좋음 (예기치 않은 변경 위험 때문)

지역 변수

  • 함수 또는 블록({ }) 내부에서 선언된 변수
  • 함수 또는 블록({ }) 안에서만 사용 가능. 즉, 함수 또는 블록({ }) 바깥에서는 접근할 수 없음
  • 따라서 같은 이름의 변수를 다른 함수 또는 블록({ })에서 중복 선언 가능

블록 스코프 let과 const

  • let과 const는 블록 스코프(Block Scope)를 가짐!
  • 즉, { } 내부에서 선언되면 해당 블록 안에서만 사용 가능.
  • 블록 내부에서 선언된 let과 const는 바깥에서 접근할 수 없음.
  • if문, for문, while문, 함수 내부 등 블록({ })을 벗어나면 사용 불가

정리

  • 전역 변수 (let, const) → 함수 바깥에서 선언하면 어디서든 사용 가능 (지양해야 함!)
  • 지역 변수 (let, const) → 함수 내부나 블록 {} 안에서 선언되면 그 범위 내에서만 사용 가능 (권장!)
  • 블록 스코프 → let, const는 {} 안에서 선언되면 바깥에서 접근 불가
  • 가능하면 const를 우선 사용하고, 필요할 때만 let을 사용하는 것이 좋다!
