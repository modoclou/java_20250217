Array

  • 변수에는 데이터를 한 개만 저장하지만
  • 배열은 하나의 변수에 데이터를 여러 개 저장할 수 있다.
  • 여러가지 자료형의 데이터를 하나의 배열 이름으로 묶어서 선언할 수 있다.
  • 배열에 있는 데이터는 인덱스 번호로 접근한다.
  • 인덱스 번호는 0부터 시작한다.
  • 배열 선언
    데이터 값을 쉼표로 구분해서 대괄호([ ])로 묶어서 선언
    대괄호([ ]) 안에 값을 입력하지 않으면 빈 배열이 만들어 진다.
    배열에 있는 데이터 접근
    배열명[인덱스 번호]
    배열의 마지막 데이터는 배열명[배열의 크기-1]
    length : 배열의 크기(배열 요소 개수)를 가져오는 속성
