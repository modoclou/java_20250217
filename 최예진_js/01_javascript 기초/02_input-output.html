간단한 입출력 방법

알림 창 출력

  • window.alert();
  • 웹 브라우저에서 알림 창을 열어 메시지 표시
  • window 객체는 브라우저의 최상위 객체이며 생략 가능하다.

확인 창에서 입력받기

  • window.confirm();
  • 사용자가 [확인] 또는 [취소] 버튼 중에서 직접 클릭할 수 있다.
  • 선택한 결과에 맞게 프로그램이 동작한다.
  • window 객체는 브라우저의 최상위 객체이며 생략 가능하다.

프롬프트 창에서 입력받기

  • window.prompt(메시지) 또는 window.prompt(메시지, 기본값);
  • 텍스트 필드가 있는 작은 창.
  • 텍스트 필드 안에 입력받은 내용을 프로그램에 사용할 수 있다.
  • window 객체는 브라우저의 최상위 객체이며 생략 가능하다.

웹 브라우저 화면에 출력

  • document.write();
  • document 객체는 HTML 문서이며 write() 메소드는 body 요소 안에 쓴다.
  • HTML 태그도 함께 사용할 수 있다.

콘솔 창 출력

  • window.console.log();
  • 브라우저의 콘솔 창에 출력한다.
  • 콘솔 창에서 소스 코드의 오류를 발견하거나 변수값을 확인할 수도 있다.
  • window 객체는 브라우저의 최상위 객체이며 생략 가능하다.
