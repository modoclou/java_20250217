switch 조건문

  • 조건식의 결과값을 case에서 찾아 일치하는 값이 있을 때 그에 해당하는 코드 실행
  • 해당하는 값이 없을 경우 default의 코드를 실행하며 default는 생략 가능하다.

switch(비교할 값) { case 비교값1 : 비교값1의 결과가 true면 이 부분의 코드 실행; break; case 비교값2 : 비교값2의 결과가 true면 이 부분의 코드 실행; break; case 비교값3 : 비교값3의 결과가 true면 이 부분의 코드 실행; break; default : 어떠한 case문에도 해당되지 않은 경우 이 부분의 코드 실행; break; }
