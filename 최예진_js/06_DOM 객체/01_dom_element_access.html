DOM(Document Object Model)이란?

DOM(문서 객체 모델, Document Object Model)은 웹 페이지의 HTML을 자바스크립트에서 조작할 수 있도록 객체 형태로 변환한 것.

  • 웹 문서를 자바스크립트에서 조작할 수 있도록 객체 형태로 변환한 모델
  • HTML 문서를 트리(Tree) 구조의 노드(Node) 객체로 표현
  • 자바스크립트를 사용하여 HTML, CSS, 이벤트 등을 변경 가능

DOM 구조

DOM은 트리(Tree) 형태의 계층적 구조로 되어 있다.

Document
HTML 문서 전체를 나타내는 객체 (document)
Element 노드
HTML 태그 하나하나를 객체로 표현 (div, p, a 등)
Attribute 노드
태그의 속성을 나타내는 객체 (class, id 등)
Text 노드
HTML 태그 내부의 텍스트 (Hello, World! 등)

DOM 요소 선택하기

아이디 접근 : getElementById()
id 값으로 요소 선택
무조건 하나만 선택
예 : document.getElementById("title")
요소명 접근 : getElementsByTagName()
태그 이름으로 여러 요소 선택
배열로 선택
예 : document.getElementsByTagName("p")
클래스명 접근 : getElementsByClassName()
클래스 값으로 여러 요소 선택
배열로 선택
예 : document.getElementsByClassName("box")
css 선택자 접근 : querySelector("css 선택자")
CSS 선택자 방식으로 요소 선택
여러 개가 있어도 첫 번째만 선택
예 : document.querySelector(".box")
같은 css 선택자 모두 접근 : querySelectorAll("css 선택자")
CSS 선택자 방식으로 여러 요소 선택
배열로 선택
예 : document.querySelectorAll(".box")
